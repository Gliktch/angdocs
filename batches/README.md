# Batches

Per-batch work logs for incomplete or resumable audit work.

Top-level `*.md` files here are active, waiting, blocked, or pending audit batches. Move completed batches to `done/`.

Each batch file should record:

* scope and target version
* source checkout/tag/commit
* text files inspected
* code/data files inspected
* model/escalation level used
* findings produced or reasons no change was needed
* verification notes
* the next step while the batch remains incomplete
